### gorchichka bot
> simple telegram bot for [gorchichka](https://gorchichka.com)

### Development

1. Create `.env` file with `BOT_TOKEN` variable
  ```sh
  BOT_TOKEN=123456foobar
  ```

2. Then run the bot locally
  ```sh
  yarn start
  ```
